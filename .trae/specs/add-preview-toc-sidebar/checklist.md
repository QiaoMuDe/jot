# Checkpoints

- [x] **TOC 侧栏 DOM 存在**：`index.html` 的 `editor-panes` 内包含 `#tocSidebar` 元素
- [x] **预览模式显示 TOC**：切换到预览模式下，TOC 侧栏和预览区左右并排显示
- [x] **非 .md 文件不显示 TOC**：打开非 Markdown 笔记时，预览模式下无 TOC 侧栏
- [x] **标题自动提取**：文档中的 H1~H6 标题被正确提取并显示在 TOC 中
- [x] **层级缩进正确**：TOC 中的标题按 H1~H6 层级正确缩进
- [x] **点击跳转**：点击 TOC 目录项，预览区平滑滚动到对应标题
- [x] **锚点 ID 正确**：渲染后的 HTML 中 `<h1>`~`<h6>` 元素包含唯一 `id`
- [x] **滚动高亮**：滚动预览内容时，TOC 中当前章节高亮
- [x] **高亮项可见**：高亮的 TOC 项自动滚动到 TOC 可视区域内
- [x] **空文档行为**：空文档或无标题文档的预览模式下，TOC 不显示
- [x] **折叠/展开**：点击折叠按钮可收起/展开 TOC 侧栏
- [x] **状态持久化**：折叠/展开状态在关闭编辑器后重新打开时恢复
- [x] **主题适配**：深色/浅色主题下 TOC 侧栏样式正确
- [x] **动画流畅**：折叠/展开动画流畅，无卡顿
- [x] **全屏模式兼容**：全屏编辑模式下 TOC 侧栏正常工作
