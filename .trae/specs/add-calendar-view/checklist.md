# 日历视图 - 验收清单

## 后端
- [ ] `GetNotesByDate(date string)` 返回指定日期非删除笔记，按 `created_at DESC` 排序
- [ ] `GetMonthNoteCounts(year, month int)` 返回某月每日笔记计数的 `map[int]int`
- [ ] 后端接口在 Wails 绑定中正确暴露，前端可调用

## 前端视图注册
- [ ] 更多菜单中存在"日历"菜单项，点击可切换到日历视图
- [ ] 日历视图有"← 返回"按钮，点击可回到笔记首页
- [ ] 视图切换动画正常，无闪烁或布局错乱

## 日历组件
- [ ] 打开日历视图默认显示当前月份
- [ ] 有笔记的日期下方显示墨水圆点（三档：1条=小淡、2-5条=中、6+=大实）
- [ ] ◀ ▶ 按钮可切换月份，圆点标记同步更新
- [ ] 点击某天，该天高亮显示（accent 圆形背景 + 白色文字）

## 日期笔记列表
- [ ] 选中日期后，右侧加载该日创建的所有笔记
- [ ] 笔记列表显示标题、笔记本名称、创建时间
- [ ] 列表有入场 stagger 动画
- [ ] 日期无笔记时显示友好的空状态

## 笔记跳转
- [ ] 点击笔记列表中的条目，能正确跳转：
  - 关闭当前编辑器（如有）
  - 切回卡片网格视图
  - 选中该笔记所在笔记本
  - 加载笔记列表
  - 打开该笔记编辑器

## 视觉与体验
- [ ] 日历视图适配全部 14 套主题，配色使用 CSS 变量
- [ ] 日历年月切换有淡入动画
- [ ] 初始化时默认选中今天，并加载今日笔记
- [ ] 笔记列表 item hover 有视觉反馈
