# 验证清单

- [x] 编辑器打开时按 Ctrl+F 显示查找条，输入框自动聚焦
- [x] 查找条包含：搜索输入框、匹配计数、上一项/下一项按钮、关闭按钮
- [x] 查找条输入内容后实时高亮 textarea 中所有匹配项
- [x] 查找计数格式正确（"当前/总数"）
- [x] 按 `[` 导航到上一个匹配项，按 `]` 导航到下一个匹配项
- [x] 当前激活匹配项与其他匹配项使用不同高亮颜色
- [x] 导航时滚动到匹配位置
- [x] 按 Ctrl+H 同时显示查找条和替换条，替换条位于查找条下方
- [x] 如果查找条已显示（Ctrl+F），再按 Ctrl+H 追加显示替换条
- [x] 查看（只读）页面按 Ctrl+F 显示查找条，替换不可用
- [x] 新建页面按 Ctrl+H 显示查找+替换条
- [x] 编辑页面 Markdown 预览模式下按 Ctrl+H 提示"请先切换到纯文本模式"
- [x] 替换单个按钮替换当前激活匹配项
- [x] 替换全部按钮替换所有匹配项
- [x] 替换后更新高亮和计数
- [x] 查找/替换输入时焦点不跳回内容区
- [x] 关闭按钮关闭对应的条
- [x] Escape 键关闭查找/替换条（若已打开）
- [x] 编辑器关闭后查找/替换条自动重置
- [x] 全屏模式下查找/替换条正常显示
- [x] 编辑器关闭时（非编辑器模式）Ctrl+F 保持原行为：聚焦顶部搜索栏
