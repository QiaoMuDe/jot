# Checkpoints

- [x] 引用笔记选择器打开时按 Enter → 触发确认引用（排除搜索框中的 Enter）
- [x] 引用笔记选择器打开时按 ESC → 浮层关闭，停留在 AI 聊天界面，不切换到笔记首页
- [x] 引用笔记选择器关闭时按 ESC → 全局 ESC 行为正常（关闭弹窗/返回首页）
- [x] 笔记编辑器中按 Ctrl+数字（如 Ctrl+6）→ 不触发全局导航
- [x] 笔记查看页面中按 Ctrl+数字 → 不触发全局导航
- [x] 新建笔记面板中按 Ctrl+数字 → 不触发全局导航
- [x] 其他页面（如设置页、数据管理）中 Ctrl+数字快捷键仍正常生效
