# 设置页卡片重组 Spec

## Why
当前设置页有 10 个设置卡片，其中快速笔记（1 个开关）、笔记排序（1 个分段控件）、分页大小（1 个分段控件）都是单人配置项的小卡片，占用过多垂直空间。AI 联网搜索也仅 2 个配置项，作为独立卡片显得单薄。将这些小卡片合并到相关卡片中，可减少视觉碎片，提升浏览效率。

## What Changes
1. **快速笔记 → 并入编辑器选项**：将"快速笔记"卡片的开关移入"编辑器选项"卡片底部，卡片标题改为"编辑器"。
2. **笔记排序 + 分页大小 → 合并为新卡片"笔记列表"**：将排序分段控件和分页分段控件放在同一卡片内，上下排列。
3. **AI 联网搜索 → 并入对话增强**：将联网搜索的 API Key 和默认开关移入"对话增强"卡片底部，卡片标题改为"对话与搜索"。

**总卡片数: 10 → 7**

## Impact
- Affected specs: 无（纯前端 UI 重组）
- Affected code:
  - `frontend/index.html` — 移动/重组 HTML 结构
  - `frontend/src/css/components/settings-panel.css` — 可能需微调布局
  - `frontend/src/main.js` — 加载顺序可能需调整

## ADDED Requirements
无新增功能，仅布局重组。

## MODIFIED Requirements
### 卡片合并
- **快速笔记配置** 将从独立卡片移入"编辑器"卡片底部，功能本身不变
- **笔记排序 + 分页大小** 合并为"笔记列表"卡片，功能不变
- **AI 联网搜索** 配置移入"对话与搜索"卡片，功能不变

## REMOVED Requirements
无功能移除。
