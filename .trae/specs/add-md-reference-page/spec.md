# Markdown 语法手册 Spec

## Why

Jot 的核心功能围绕 Markdown 展开。提供内置的 Markdown 语法手册，既能帮助不熟悉 MD 语法的用户快速上手，又能作为随时查阅的速查手册，降低新用户的学习成本，避免他们因不会 MD 语法而放弃使用编辑器。

## What Changes

### 新增：更多菜单 → "Markdown 语法手册"

- 在「更多」下拉菜单的「帮助」之后新增菜单项
- 数据动作 `data-action="md-ref"`
- 快捷键 `Ctrl+8`

### 新增：视图 `#viewMdRef`

- 遵循现有视图模式：`.view` → `.view-header` + `.scrollable-content`
- 标题：**Markdown 语法手册**
- 返回按钮 → `switchView('grid')`

### 新增：Markdown 分类卡片式展示

以卡片分区的形式展示以下语法类别，每个类别包含「语法说明」+「示例代码块」+「实时渲染预览」三段式布局：

1. **标题 (Headings)** — `#` ~ `######`
2. **文本样式** — 加粗 `**`、斜体 `*`、删除线 `~~`、行内代码 `` ` ``
3. **链接与图片** — `[title](url)`、`![alt](src)`
4. **列表** — 无序 `-`、有序 `1.`
5. **代码块** — 围栏代码块 `~~~` (支持语法高亮)
6. **引用** — `>` 块引用
7. **表格** — 标准 Markdown 表格
8. **任务列表** — `- [ ]` / `- [x]`
9. **分割线** — `---`
10. **转义字符** — `\` 特殊字符转义

每个卡片包含：
- 左上角标签（语法类别名称）
- 左侧：语法源码（带深色背景 + 行内复制按钮）
- 右侧：实时渲染效果（使用 Jot 相同的 marked + highlight.js 渲染引擎）
- 卡片右下角标注「支持快捷键 Ctrl+B / Ctrl+I」等提示

### 运行机制

- 页面为纯静态展示，不涉及后端数据
- Markdown 渲染复用现有的 `mdToHtml()` 函数
- 代码高亮复用现有的 `highlightCode()` 逻辑

## Design Requirements

### 视觉方向

采用「**教科书式卡片**」设计风格——干净、有序、可读性强：

- **色调**：主色沿用 Jot 现有的暖色调基调（琥珀色 `#D97706`），配合浅灰/暖白背景
- **卡片**：白色底色 + 圆角 `12px` + 柔和阴影，每个语法分类为一张独立卡片
- **字体**：源码区使用等宽字体（SF Mono / JetBrains Mono），预览区使用正文字体
- **布局**：两栏对比布局（源码 | 预览），源码区深色半透明背景，预览区浅色背景
- **间距**：卡片间 `24px` 间距，内边距 `20px`，呼吸感充足
- **响应**：页面整体可垂直滚动，卡片宽度自适应

### 交互细节

- 每个源码块右上角有**复制按钮**，hover 显示「复制」，点击后短暂显示「已复制 ✓」
- 复制按钮使用 JavaScript 定位，与表格复制按钮模式一致
- 卡片悬停时有轻微阴影加深的过渡效果
- 页面进入时卡片依次淡入（stagger，延迟 50ms × index）
