# 日历视图添加"回到今天"按钮

## 需求

在笔记日历视图中添加一个"回到今天"按钮，仅在用户查看**非今天**的日期时显示，点击后切回今天。

## 当前状态分析

日历视图布局（`frontend/index.html`）：

```
.view-header (← 返回 | 笔记日历)
└── .calendar-layout (flex row)
    ├── .calendar-sidebar (左侧，340px)
    │   ├── .calendar-header (← 月份 | 2026年 7月 | →)
    │   └── .calendar-grid (7×6 日期网格)
    └── .calendar-notes-panel (右侧，flex: 1)
        ├── .calendar-notes-header
        │   └── #calendarDateLabel ("7月23日 星期三")
        ├── .calendar-notes-list (笔记条目)
        └── .calendar-notes-empty (空状态)
```

关键函数（`frontend/src/js/calendar.js`）：

* `renderCalendar(year, month)` — 渲染网格，默认选中今天

* `selectDate(el, year, month, day)` — 高亮日期 + 更新日期标签 + 加载笔记

## 变更方案

### 按钮位置

放在右侧面板的 `.calendar-notes-header` 中，日期标签 `#calendarDateLabel` 旁边。

**理由**：用户查看某个非今天日期的笔记时，按钮就在日期旁边，上下文自然；比放在左侧栏底部更易发现。

### 显示条件

只在当前选中的**年月日**不是**今天**时才显示。具体判断时机：

1. `selectDate()` 完成选中后 — 最直接的判断点，每次点击日期都触发
2. 初始渲染（`renderCalendar()` 中默认选中今天时）— 默认隐藏

### 行为

点击后重新渲染当前月份（保证回到今天的月份），并选中今天的日期。

### 涉及文件及修改

#### 1. `frontend/index.html`

在 `.calendar-notes-header` 中追加一个按钮：

```html
<div class="calendar-notes-header">
    <span class="calendar-date-label" id="calendarDateLabel">选择日期查看笔记</span>
    <button class="calendar-today-btn" id="calendarTodayBtn" style="display:none;">回到今天</button>
</div>
```

#### 2. `frontend/src/css/components/calendar.css`

新增按钮样式：

* `.calendar-notes-header` 改为 `display: flex; align-items: center; justify-content: space-between;` 以容纳按钮

* `.calendar-today-btn` — 紧凑按钮样式：小字号、圆角、hover 效果

* `.calendar-today-btn` 初始隐藏（`display: none`），由 JS 控制显隐

#### 3. `frontend/src/js/calendar.js`

* 在 `initCalendarView()` 中获取 `calendarTodayBtn` DOM 引用并绑定点击事件

* 新增 `updateTodayBtnVisibility(year, month, day)` 函数：判断年月日是否等于今天，控制按钮显隐

* 在 `selectDate()` 末尾调用 `updateTodayBtnVisibility()`

* 按钮点击回调：调用 `renderCalendar()` 回到当前月份，选中今天

### 判今天逻辑

```js
function updateTodayBtnVisibility(year, month, day) {
    const now = new Date();
    const isToday = (year === now.getFullYear() && month === now.getMonth() + 1 && day === now.getDate());
    calendarTodayBtn.style.display = isToday ? 'none' : '';
}
```

### 点击按钮行为

点击"回到今天"时，需要：

1. 跳回当前月份：`renderCalendar(todayYear, todayMonth)`
2. `renderCalendar` 内部会自动选中今天（已有逻辑：首次渲染时标记 `selectedDayEl`）
3. 同时关闭按钮（`updateTodayBtnVisibility` 在 `selectDate` 中被调用）

### 边界情况

* **跨年跨月**：用户导航到 2025 年 12 月或 2027 年 1 月时，点击"回到今天"返回今月

* **首次进入日历**：默认选中今天，按钮隐藏（正确）

* **切换年份/月份后**：prev/next 按钮触发 `renderCalendar()`，后者默认选中今天 → 按钮隐藏，但用户可能点了其他日期 → 按钮显示

## 验证步骤

1. 打开日历视图 → "回到今天"按钮不显示
2. 点击非今天日期 → 按钮出现
3. 切换到其他月份后点击某日期 → 按钮出现
4. 点击"回到今天"按钮 → 回到今月，选中今天，按钮消失
5. 点击今天日期 → 按钮继续保持隐藏
