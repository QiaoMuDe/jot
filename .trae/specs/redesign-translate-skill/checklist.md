# Checklist

- [x] 翻译在「更多技能」中作为普通菜单项，点击直接激活（无子菜单展开）
- [x] 翻译 chip 显示为 `[🌐] [English] ⇄ [Chinese] [×]` 格式
- [x] 点击 chip 上的语言标签弹出语言选择浮层（含 10 种语言）
- [x] 浮层中当前语言有选中态（高亮 + ✓）
- [x] 选择与另一侧相同的语言时，自动交换两侧语言
- [x] 点击浮层外部区域关闭浮层
- [x] 点击 chip 的 × 按钮取消翻译技能
- [x] 加载旧格式会话配置（`direction: 'to_chinese'`）时自动迁移为新格式
- [x] 后端通用翻译 prompt 根据 source/target 动态生成，不区分 `skill_translate_cn/en`
- [x] 翻译技能与其他技能（如编程、写作）共存时互斥（激活翻译自动取消其他技能）
