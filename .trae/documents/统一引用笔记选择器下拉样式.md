# 统一引用笔记选择器下拉样式计划

## 当前状态

**笔记本下拉** (`#aiNoteRefNotebook`)：使用原生 `<select>` 元素，样式由 `.ai-note-ref-notebook-select` 控制。不同浏览器/平台外观不一致，且与自定义标签下拉的风格不统一。

**标签下拉** (`.ai-note-ref-filter-dropdown`)：使用自定义 button + div 面板构建，有 hover/选中态等丰富交互。但缺少自定义滚动条样式，导致在某些环境中滚动条不可见或样式异常。

## 目标

1. **笔记本下拉 → 自定义下拉**：将原生 `<select>` 重构为与标签下拉相同的自定义 button + dropdown 结构，保持视觉一致。
2. **修复标签下拉滚动条**：为标签下拉添加自定义滚动条样式（参考侧边栏 `.ai-session-list` 的细窄边缘滚动条）。
3. **两个下拉保持一致的样式**：共用同一套 CSS 变量和设计模式，仅在交互细节上有所区分。

## 修改清单

### 1. `frontend/index.html`

**替换笔记本下拉 DOM**（第 1005-1007 行）：

将：

```html
<select id="aiNoteRefNotebook" class="ai-note-ref-notebook-select">
    <option value="0">全部笔记本</option>
</select>
```

替换为：

```html
<div class="ai-note-ref-filter" id="aiNoteRefNotebookFilter">
    <button class="ai-note-ref-filter-btn" id="aiNoteRefNotebookBtn">
        <span id="aiNoteRefNotebookLabel">全部笔记本</span>
        <svg class="chevron" width="12" height="12" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round"><polyline points="6 9 12 15 18 9"/></svg>
    </button>
    <div class="ai-note-ref-filter-dropdown" id="aiNoteRefNotebookDropdown"></div>
</div>
```

结构说明：

* 外层容器 `div.ai-note-ref-filter`（与标签下拉相同）

* 触发按钮 `button.ai-note-ref-filter-btn`（与标签下拉相同类名，共用样式）

* Chevron 图标（与标签下拉相同 SVG）

* 下拉面板 `div.ai-note-ref-filter-dropdown`（与标签下拉相同类名，共用样式）

#### 3d. 修改 `rebuildNotebookOptions()` → `rebuildRefNotebookOptions()`

将函数重命名为 `rebuildRefNotebookOptions`（与标签 `renderRefTagDropdown` 命名风格一致）并重写为：

```javascript
function rebuildRefNotebookOptions() {
    if (!refNotebookDropdown || !_notebooksCache) return;
    const currentId = _currentNotebookId || 0;  // _currentNotebookId 为新增变量
    let html = '';
    html += `<div class="ai-note-ref-filter-option${currentId === 0 ? ' selected' : ''}" data-notebook-id="0">全部笔记本</div>`;
    _notebooksCache.forEach(n => {
        const selected = currentId === n.id;
        html += `<div class="ai-note-ref-filter-option${selected ? ' selected' : ''}" data-notebook-id="${n.id}">${n.name}</div>`;
    });
    refNotebookDropdown.innerHTML = html;

    refNotebookDropdown.querySelectorAll('.ai-note-ref-filter-option').forEach(opt => {
        opt.addEventListener('click', (e) => {
            e.stopPropagation();
            const notebookId = parseInt(opt.dataset.notebookId);
            _currentNotebookId = notebookId;
            // 更新按钮文字
            if (refNotebookLabel) {
                if (notebookId === 0) refNotebookLabel.textContent = '全部笔记本';
                else {
                    const nb = (_notebooksCache || []).find(n => n.id === notebookId);
                    refNotebookLabel.textContent = nb ? nb.name : '全部笔记本';
                }
            }
            // 关闭下拉
            if (refNotebookFilter) refNotebookFilter.classList.remove('open');
            // 更新选中样式
            refNotebookFilter.classList.toggle('active', notebookId > 0);
            // 触发列表刷新
            loadNoteList(false);
        });
    });
}
```

## 交互行为

* **互斥展开**：笔记本下拉和标签下拉不会同时展开，点击一个会关闭另一个

* **点击外部关闭**：点击浮层外任意区域自动关闭展开的下拉

* **选中态**：选中具体笔记本时按钮显示 `.active` 样式（accent 色），"全部笔记本"时无 active

* **全选联动**：`_currentNotebookId` 变化 → `_refSelectAll` 重置（已有逻辑）

## 验证步骤

1. **笔记本下拉功能**：点击笔记本按钮 → 展开下拉列表 → 选择一个笔记本 → 列表按该笔记本筛选 → 按钮显示笔记本名称 + active 色
2. **标签下拉功能**：点击标签按钮 → 展开下拉列表 → 选择一个或多个标签 → 列表按标签筛选（已有功能不受影响）
3. **互斥展开**：点击笔记本下拉 → 展开；再点击标签下拉 → 笔记本下拉关闭，标签下拉展开
4. **点击外部关闭**：展开任一下拉 → 点击浮层外部 → 下拉关闭
5. **滚动条可见性**：笔记本下拉/标签下拉中选项较多时 → 滚动条细窄可见，悬停时颜色加深
6. **全选联动**：选择笔记本 → 全选按钮状态重置（已有功能不受影响）
