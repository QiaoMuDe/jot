# 日历笔记改为原地打开查看模式

## 一、变更概要

将日历笔记列表中点击笔记的行为，从"切换到网格视图并打开编辑器"改为"在原地（日历视图）直接打开编辑器浮层查看模式"。

## 二、当前状态分析

### 当前行为

`openNoteFromCalendar()`（[calendar.js](frontend/src/js/calendar.js#L262-L286)）执行以下步骤：

1. 关闭已有编辑器（若有）
2. `switchView('grid')` — 切换到网格视图
3. 切换左侧笔记本高亮到该笔记所属笔记本
4. `await window.loadNotes()` — 重新加载网格视图笔记列表
5. `window.openEditor(note.id, true)` — 以只读模式打开编辑器

### 关键发现

* `#editorOverlay` 是 `position: fixed` 浮层（`z-index: 1000`），不依赖任何视图

* `closeEditor()` 只做退出动画和清理，不做视图切换

* `switchView('calendar')` 会调用 `refreshCalendarView()` 重新渲染日历（但改为原地打开后不需要触发）

* 日历状态变量（`calendarYear`、`calendarMonth`、选中日期、笔记列表）在视图不切换时自然保留

## 三、改动方案

### 修改文件

**仅需修改 1 个文件**：[calendar.js](frontend/src/js/calendar.js)

### 具体改动

#### `openNoteFromCalendar()` 函数

**旧逻辑**：

```javascript
async function openNoteFromCalendar(note) {
    // 仅在编辑器实际打开时关闭
    const viewEditor = document.getElementById('viewEditor');
    if (viewEditor && viewEditor.classList.contains('active')) {
        const closeEditorSafe = getCloseEditorSafe();
        if (closeEditorSafe) await closeEditorSafe();
    }

    window.switchView('grid');               // ① 切换到网格视图

    if (note.notebook_id) {
        window.state.activeNotebookId = note.notebook_id;
        if (typeof window.updateSidebarMenuItem === 'function') {
            window.updateSidebarMenuItem(note.notebook_id);  // ② 切换笔记本
        }
    }

    await window.loadNotes();                 // ③ 重新加载笔记列表
    window.openEditor(note.id, true);         // ④ 打开编辑器
}
```

**新逻辑**：

```javascript
async function openNoteFromCalendar(note) {
    // 仅在编辑器实际打开时关闭
    const viewEditor = document.getElementById('viewEditor');
    if (viewEditor && viewEditor.classList.contains('active')) {
        const closeEditorSafe = getCloseEditorSafe();
        if (closeEditorSafe) await closeEditorSafe();
    }

    // 直接在日历视图内打开编辑器查看模式
    window.openEditor(note.id, true);
}
```

### 改动说明

| 步骤                          | 旧行为      | 新行为        | 原因              |
| --------------------------- | -------- | ---------- | --------------- |
| 关闭编辑器                       | ✅ 保留     | ✅ 保留       | 防竞态             |
| `switchView('grid')`        | 切换到网格视图  | ❌ 删除       | 编辑器是独立浮层，无需切换视图 |
| 切换笔记本高亮                     | 同步侧边栏笔记本 | ❌ 删除       | 日历视图下不显示笔记本侧边栏  |
| `loadNotes()`               | 重新加载网格列表 | ❌ 删除       | 不切换到网格视图，无需加载   |
| `openEditor(note.id, true)` | 在网格视图上打开 | ✅ 保留（直接调用） | 编辑器浮层不依赖视图      |

## 四、影响范围

* **无副作用**：编辑器是独立浮层，其他视图不受影响

* **日历状态保留**：关闭编辑器后，日历视图的年月、选中日期、笔记列表滚动位置均保持原样

* **笔记本侧边栏**：日历视图下笔记本侧边栏已自动折叠，切换笔记本高亮无意义

## 五、验证方式

1. 点击日历中的笔记 → 应直接在当前日历视图上打开编辑器浮层，而不是跳到网格视图
2. 关闭编辑器浮层 → 应回到日历视图，年月、选中日期、笔记列表不变
3. 从其他视图打开编辑器 → 行为应不受影响
